User Manual

  1. Enter coordinates in Easting, Northing format, one point per line.
  2. The drawing and calculations update automatically as you type.
  3. Use Refresh to force a redraw.
  4. Use Share to send the drawing image + results to WhatsApp, email, or any app.
  5. Use the download icon to save the plot as a PNG image.
  6. Use Clear to reset everything.

Minimum 3 points required to draw a polygon.
